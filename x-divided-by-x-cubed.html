[tex] - 27[/tex]

[tex]( x + 2)( {x}^{2} - 4x + 12) = {x}^{3} - {2x}^{2} + 4x + 24[/tex]

then add the remainder:

[tex] {x}^{3} - {2x}^{2} + 4x + 24 + ( - 27)= ^{3} - {2x}^{2} + 4x - 3[/tex]

so, the remainder is:

[tex] - 27[/tex]
