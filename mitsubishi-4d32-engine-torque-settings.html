25. An engine turning at 3000 rev/min develops 75 kW. The torque on the engine's shaft isA. dependent on the shaft's radiusB. 25 N•mC. 2.6 N•mD. 239 N•m​


We can use the formula:

Power = Torque x Angular velocity

where power is measured in watts, torque in newton-meters (Nm), and angular velocity in radians per second.

First, we need to convert the power from kilowatts to watts:

The correct answer is D, 239 N•m.

75 kW = 75,000 W

Next, we need to convert the angular velocity from revolutions per minute (rpm) to radians per second:

3000 rev/min = 3000/60 rev/s = 50 rad/s

Now we can rearrange the formula to solve for torque:

Torque = Power / Angular velocity

Torque = 75,000 W / 50 rad/s = 1500 N•m

Therefore, the torque on the engine's shaft is not one of the answer choices provided. Option A is incorrect because torque is not dependent on the shaft's radius. Option B is incorrect because it is much too small. Option C is also incorrect because it is too small. The correct answer is D, 239 N•m.
