4. 2. 98, 95, 96, 93, 98, 97, 98 Mean: Median: Mode:​


Answer:

Mean is the average, median is the middle number, mode is the most frequent number.

To find mean, we need to add all of the numbers and divide it base on how many many data are there

To find median, we need to sort it in ascending order, lowest to highest. Then we will pick the number in the very middle as median. If there are even number of data, we need to add the two median and divide it to 2.

To find mode, we need to tally up or observe what is the most frequent number appearing in the data.

Mean:

98+ 95+ 96+ 93+ 98+ 97+ 98= 675 / 7 ( bcoz there are 7 data) = 96.43

Median:

98, 95, 96, 93, 98, 97, 98 => 93, 95, 96, 97, 98, 98, 98 so the median is 97

Mode:

98, 95, 96, 93, 98, 97, 98 => as we can see, there are 3 98s and it is the most frequent number so the mode is 98
