9. Estimate the market value of the land having the total area of A(x)=20x²+20x+100 square meters, you will compute the total area of the land where in every square meter is worth Php 1,000​


Step-by-step explanation:

The total area of the land is given by A(x) = 20x² + 20x + 100 square meters. If every square meter is worth Php 1,000, then the market value of the land is:

Market value = A(x) x Php 1,000

Substituting A(x), we get:

Market value = (20x² + 20x + 100) x Php 1,000

Simplifying, we get:

Market value = 20,000x² + 20,000x + 100,000

Therefore, the market value of the land is a quadratic function of x, and we need to know the value of x to compute it. If x is in meters, then we can use the quadratic formula to find the value of x that maximizes the market value:

x = -b / 2a

where a = 20,000 and b = 20,000.

Substituting the values of a and b, we get:

x = -20,000 / 2(20,000) = -0.5

Since x is a length, it cannot be negative. Therefore, we conclude that the market value of the land is maximized at the vertex of the parabola, which has an x-coordinate of -b / 2a = -0.5. Substituting this value of x into the equation for the market value, we get:

Market value = 20,000(-0.5)² + 20,000(-0.5) + 100,000

Market value = 105,000

Therefore, the estimated market value of the land is Php 105,000.
