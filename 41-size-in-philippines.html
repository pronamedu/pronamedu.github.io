12. Find the sample size given 99% confidence, Margin of Error = 0.18 and ̂p =0.30. Select one: a. 41 b. 42 c. 43 d. 44 #oed


Answer:

The formula for calculating sample size for a proportion is:

n = (z^2 * p * q) / E^2

Where:

- z is the z-score corresponding to the desired confidence level (in this case, 2.576 for 99% confidence)

- p is the estimated proportion

- q is 1-p

- E is the margin of error

Plugging in the given values, we get:

n = (2.576^2 * 0.3 * 0.7) / 0.18^2

n ≈ 42.71

Since we can't have a fraction of a person in a sample, we round up to the nearest whole number to get:

n ≈ 43

Therefore, the answer is (c) 43.
