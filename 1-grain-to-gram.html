If you are going to fill up all the squares in a chessboard. By filling up the first square with one grain of rice, 2 grains on the second square and four grains on the next square.

Find for the total number of grains to occupy all the squares in a chessboard

Solution :

With 64 squares on a chessboard, if the number of grains doubles on successive squares, then the sum of grains on all 64 squares is: 1 + 2 + 4 + 8 + ... and so forth for the 64 squares.

Formula for this is

(2 raise to 64 ) - 1

x = (2^64)-1

x = 1.8447 x 10^19
