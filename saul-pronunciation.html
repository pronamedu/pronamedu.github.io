3.125 = 3 + 0.125

3 is the hours, so we have 3 hours. Now we have to convert 0.125 hours to minutes and seconds, to do this we need to multiply the value by 60

0.125(60) = 7.5

We have 7 as the number of minutes and 0.5 as the number of seconds, now we have to convert 0.5 minutes to seconds, we can do this by multiplying it by 60

0.5(60) = 30

And now we have the last time unit we need which is seconds, now we just need to combine all of them to identify the exact time needed for Saul to travel
