Ⓒ what percent of the volleyball player's display a height between 166 cm and 170 cm? 6 what percent of the volleyball players display a height between 170 cm and 172 cm ? 6 what percent of the volleyball players display a height less than 170 cm ?​


To solve this problem, we will use the properties of the normal distribution and the standard normal distribution table.

a) To find the percent of volleyball players with a height between 166 cm and 170 cm, we need to find the area under the normal curve between these two values. We can standardize the values using the formula:

z = (x - μ) / σ

where z is the z-score, x is the given height, μ is the mean height, and σ is the standard deviation.

So for x = 166 cm, we have:

z = (166 - 168) / 2 = -1

For x = 170 cm, we have:

z = (170 - 168) / 2 = 1

Using the standard normal distribution table, we can find the area between -1 and 1 to be approximately 0.6827.

Therefore, about 68.27% of the volleyball players display a height between 166 cm and 170 cm.

b) To find the percent of volleyball players with a height between 170 cm and 172 cm, we can repeat the same process:

For x = 170 cm, we have:

z = (170 - 168) / 2 = 1

For x = 172 cm, we have:

z = (172 - 168) / 2 = 2

Using the standard normal distribution table, we can find the area between 1 and 2 to be approximately 0.1359.

Therefore, about 13.59% of the volleyball players display a height between 170 cm and 172 cm.

c) To find the percent of volleyball players with a height less than 170 cm, we need to find the area under the normal curve to the left of 170 cm.

For x = 170 cm, we have:

z = (170 - 168) / 2 = 1

Using the standard normal distribution table, we can find the area to the left of 1 to be approximately 0.8413.

Therefore, about 84.13% of the volleyball players display a height less than 170 cm.
