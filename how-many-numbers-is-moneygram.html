3. The total number of goats and cows in the farm is 335. If the number of cows is two-thirds of the number of goats, how many cows are there in the farm?


Answer:

There are 134 cows in the farm.


Step-by-step explanation:

Let's use algebra to solve this problem.

Let's assume that the number of goats in the farm is "G", and the number of cows is "C". From the problem statement, we know that:

G + C = 335 (the total number of goats and cows is 335)

C = (2/3)G (the number of cows is two-thirds of the number of goats)


We can use the second equation to express "G" in terms of "C" by multiplying both sides by 3/2:

3/2 * C = (3/2) * (2/3) * G

3/2 * C = G

Now we can substitute this expression for "G" into the first equation:

G + C = 335

3/2 * C + C = 335

5/2 * C = 335

C = 67 * 2/5

16. a four digit is to be composed from digits 0,1,2,3,4,5,6,7 (repetition of digit is allowed and no number may begin with 0 )How many ways can the composition be done?How many of these numbers are divisible by 5?How many of these numbers are less than 4000?How many of these numbers are even?​


Answers:

How many ways can the composition be done?

3584

How many of these numbers are divisible by 5?

896

How many of these numbers are less than 4000?

1536

How many of these numbers are even?

1792  

Step-by-step explanation:

How many ways can the composition be done?

Since 0 cannot be the thousands digit, there are only 8 - 1 = 7 possible digits to be selected to put on thousands digit.

Since the hundreds digit has no restrictions, all of the digits can be selected to put on hundreds digit.

Likewise, tens and ones digit has no restrictions, therefore all of the digits can be selected to put on tens and ones digit.

Hence, there are 7 × 8 × 8 × 8 = 3584 ways that the composition can be done.

How many of these numbers are divisible by 5?

Since 0 cannot be the thousands digit, there are only 8 - 1 = 7 possible ways to select on thousands digit.

Since the hundreds digit has no restrictions, all of the digits can be selected to put on hundreds digit.

Likewise, tens digit has no restrictions, therefore all of the digits can be selected to put on tens digit.

Now, according to the divisibility rule, a number is divisible by 5 if the ones digit is either 0 or 5.

Therefore, there are only 2 possible digits that can be selected to put on the ones digit.

Hence, there are 7 × 8 × 8 × 2 = 896 numbers that are divisible by 5

How many of these numbers are less than 4000?

If the numbers are less than 4000, then the thousands digit must be less than 4.

Therefore, there are 8 - 4 - 1 = 3 possible digits that we can select to put on thousands digit.

We can select any digits to put on hundreds, tens and ones digit since it will be always less than 4000.

Hence, there are 3 × 8 × 8 × 8 = 1536 numbers that are less than 4000.

How many of these numbers are even?​

Since 0 cannot be the thousands digit, there are only 8 - 1 = 7 possible digits to be selected and put on thousands digit.

Since the hundreds digit has no restrictions, all of the digits can be selected to put on hundreds digit.

Likewise, tens digit has no restrictions, therefore all of the digits can be selected to put on tens digit.

Now, to be a number be even, it must end with an even digit.

Since there are 4 even digits i.e., 0, 2, 4 and 6, then there are 4 possible ways to select digits on tens digit.

Hence, there are 7 × 8 × 8 × 4 = 1792 even numbers.
