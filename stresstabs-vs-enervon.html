3. Is it safe to take stresstabs and myra e at the same time?


It is generally safe to take StressTabs and Myra-E at the same time. StressTabs is a multivitamin while Myra-E is a Vitamin E supplement. So long as there is no overdose of either of the two then you should be safe. Also consider if you have any allergic to the components of any of the two mentioned drugs. Should you experience any difficulty in breathing, rashes or any inflammation, stop the drug and consult a doctor right away.  

Why do we take vitamin supplements?

We take vitamin supplements because there are certain nutrients that are not produced by the body nor can they be easily sourced from food. Eating a healthy diet of fruits and green leafy vegetables is still the best way to get our vitamins and minerals but in many instances, we are not able to maintain this, hence we take vitamin supplements.

Take note that before taking any kinds of drugs, including vitamin and food supplements, it is best to consult a doctor for guidance. Also let the doctor know of any other medicines you are taking as certain medicines are known to interact with each other, causing harm to the body. So far, by experience, StressTabs and Myra-E have no negative effects--but you can never be too careful.

18. explain the difference of JPEG vs. GIF vs. PNG vs. BMP vs. TIFF


JPEG

(Joint Photographic Experts Group)

is a File Format with a lossy compression. It allows some data loss for a smaller file size. It compresses grayscale and color information but allows for images to be displayed in 16,000,000 colors.

GIF

(Graphics Interchange Format)

is a File Format with a lossless compression. Meaning, there is no data lost in the compression. GIF supports animation and is preferred for images with only a few colors. (GIF only allows for 256 colors)

PNG

(Portable Network Graphics)

is a File Format that takes off from the transparency capabilities of GIF, primarily developed due to GIF's licensing requirement.

TIFF

(Tagged Image File Format)

is a File Format that is a flexible bitmap image that is supported by many software. It has a much larger file size compared to JPEG and GIF and is commonly recommended by professional printers since it is supported by most press machines.

BMP

(Bitmap)

The standrad Windows file format on DOS and Windows–compatible platforms. It has a lossless compression.

23. conflictcharacter vs charactercharacter vs selfcharacter vs naturecharacter vs society​

I'm not sure with the answer that I'm supposed to give so I'll just briefly explain the 4 conflicts

Explanation:

character vs character:

this is when the character's problem/conflict is due to another character (ex.: the main character got in a fight with another character and that's the main problem/conflict in the text)

character vs self:

this is when the character's problem/conflict revolves around their state of mind.(ex.: internal conflicts, psychological, the conflict happens inside their head)

character vs nature:

this is when the conflict was not caused by another character or by themselves but rather, by nature or environment (ex.: natural disasters)

character vs society:

this is when society or the people around the character are the ones causing the conflict. (ex: prejudice of society toward the character, actions or ideas that the character has but society can't accept).
