8. How many three-digit numbers are there, each of which has exactly 11 tens, and when added to 90 results in a number smaller than 203?​


Answer:

13

Step-by-step explanation:

We know that a three-digit number has the form "abc" where a is the hundreds digit, b is the tens digit, and c is the units digit.

From the information given we know that b is equal to 1, thus the only variable digit is c, and we know that the number needs to be less than 203 when added to 90.

So we can write the inequality as:

abc + 90 < 203

abc < 113

Now we need to find how many values c can take. c is the units digit and can take the value between 0 and 9.

so we have 0 < c < 13 (0 <= c < 13)

Therefore, there are 13 - 0 = 13 possible values for c.

So the total number of three-digit numbers that have exactly 11 tens and when added to 90 results in a number smaller than 203 is 13.
