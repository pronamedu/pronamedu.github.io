Answer:

Activity 4A:

To compute for the Room Occupancy Percentage (ROP) of Amaya Hotel and Restaurant, we need to know the total number of rooms in the hotel and the number of occupied rooms.

Total number of rooms = 4 (Rooms 101, 102, 201, and 202)

Number of occupied rooms = 3 (Rooms 101, 201, and 202)

Therefore, ROP = (number of occupied rooms / total number of rooms) x 100%

ROP = (3 / 4) x 100%

ROP = 75%

So the Room Occupancy Percentage of Amaya Hotel and Restaurant is 75%.

Activity 4B:

To compute for the Total Room Sales (TRS) of Room 101 in Nocete's Hotel, we need to know the rate per room and the number of times it was occupied.

Rate per room = Php 350.00

Number of times Room 101 was occupied = 3

Therefore, TRS = rate per room x number of times occupied

TRS = Php 350.00 x 3

TRS = Php 1050.00

So the Total Room Sales of Room 101 in Nocete's Hotel is Php 1050.00.

Activity 5:

To compute for the Total Room Sales (TRS) of Shine Hotel, we need to know the room rate, the number of rooms sold, and the floor number of the rooms sold.

Here are the data provided:

(Floor) (No. of Rooms Sold) (Room Rate)

2 1 Php 1050.00

3 2 Php 2100.00

4 1 Php 2780.00

To compute for the TRS, we need to multiply the room rate by the number of rooms sold and sum them up.

TRS = (number of rooms sold on floor 2 x room rate) + (number of rooms sold on floor 3 x room rate) + (number of rooms sold on floor 4 x room rate)

TRS = (1 x Php 1050.00) + (2 x Php 2100.00) + (1 x Php 2780.00)

TRS = Php 8030.00

Therefore, the Total Room Sales of Shine Hotel is Php 8030.00.
