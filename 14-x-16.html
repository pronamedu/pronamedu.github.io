6. find the value of X in the proportion 8:X=16:24 a.10 b.12 c.14 d.16​


Given proportion: 8:X = 16:24

The proportion can be written as

[tex]\frac{8}{X} = \frac{16}{24}[/tex]

Two ratios are equal, so we can use the cross-multiplication method to solve for the unknown.

By cross-multiplication,

[tex]8 * 24 = 16X[/tex]

[tex]192 = 16X[/tex]

[tex]X = 192 \div 16[/tex]

[tex]\boxed{X = 12}[/tex]

Hence, option b is the correct answer.
