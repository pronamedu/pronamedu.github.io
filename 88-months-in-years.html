To find the average occupancy percentage for the whole year, we need to consider the occupancy percentages for both the summer months and the rest of the year.

Let O1 be the occupancy percentage for the summer months, which is 88%.
Let O2 be the occupancy percentage for the rest of the year, which is 44%.

To calculate the average occupancy percentage for the whole year, we can use the formula:

Average occupancy percentage = (total occupancy percentage for the year) / (number of months in the year)

Since there are 12 months in a year, we can calculate the total occupancy percentage for the year as follows:

Total occupancy percentage for the year = (O1 * 3 months) + (O2 * 9 months)
= (88% * 3) + (44% * 9)
= 264% + 396%
= 660%

Therefore, the average occupancy percentage for the whole year is:

Average occupancy percentage = Total occupancy percentage for the year / number of months in the year
= 660% / 12
= 55%
