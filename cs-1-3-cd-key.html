If you see a message to press any key to boot the CD and you fail to do so, what will happen next is the computer will boot or start you currently installed version of your operating system (Example: Windows 10, Windows 8, Windows 2000 server etc.)

The message in a command prompt style is a message by the bootable device (like CD or USB) that the computer will be booting the said removable devices. You need to boot to these devices if you will reinstall the operating system or you will reformat and reinstall the operating system. Once you have inserted the media and restart performed, this message will be shown.
