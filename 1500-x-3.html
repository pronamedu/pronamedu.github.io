11. hazel's daily profit for selling books is described by the formula P= -x²+80x - 1500. What should be the value of x if her profit is positive​


Answer:

30 < x < 50

Step-by-step explanation:

Given the function:

P = -x²+80x-1500

Inorder to get the positive profit , the function should be greater than zero because all the numbers greater than zero are positive.

P = -x²+80x-1500

will turn it into

-x²+80x-1500 > 0

Now , solve for the values of x

-x²+80x-1500 > 0

(-x+50)(x-30) > 0

(-x-50) > 0 and (x-30) > 0

-x > 50 and x > 30

x < 50

Hence , the solutions will be

30 < x < 50
