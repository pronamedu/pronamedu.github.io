25. 151/3 divide by 23/18​


Answer: 39 9/23

Explanation:

Dividing two fractions is the same as multiplying the first fraction by the reciprocal (inverse) of the second fraction.

Like this, 151/3 ÷ 23/18

For fraction multiplication, multiply the numerators and then multiply the denominators to get: [tex]\frac{151*18}{3*23}[/tex] = [tex]\frac{2718}{69}[/tex]

This fraction can be reduced by dividing both the numerator and denominator by the Greatest Common Factor of 2718 and 69 using

GCF(2718,69) = 3

[tex]\frac{2718/3}{69/3}[/tex] = [tex]\frac{906}{23}[/tex]

Convert to a mixed number using

long division for 906 ÷ 23 = 39R9,  so

[tex]\frac{906}{23}[/tex] = 39 9/23

Therefore: 151/3 ÷ 23/18 = 39 9/23
