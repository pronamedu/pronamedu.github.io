•Factor x² - 5x + 6 using the AC method.

→(x - 3)(x - 2) = 0

→ x - 3 = 0

x - 2 = 0

•Set x - 3 equal to 0 and solve for x

→x - 3 + 3 = 0 + 3

•Set x - 2 equal to 0 and solve for x

→x - 2 + 2 = 0 + 2

x = 3
