12. A subdivision's water department charges 75 centavos per liter, plus a fixed charge for the first 100 L-use. Mr. Bernardo's water bill for the use of 120 L was 240 pesos. Find the fixed charge.


Charge per liter: 0.75 centavos

Bill for 120-liter consumption: 240 pesos

Fixed charge for first 100-liter consumption: C

Derive a function where x = water consumption in liters

f(x) = C + 0.75 (x - 100)   where x > 100

If the consumption/use for the month is 120 liters, then x = 120.

Solve for the fixed charge, C:

240 = C + 0.75 (120 - 100)

240 = C + 0.75 (20)

240 = C + 15

C = 240 - 15

C = 225

ANSWER: The fixed charge is 225 pesos.

Check:

f(120) = 225 + 0.75 (120-100)

f(120) = 225 + 0.75 (20)

f(120) = 225 + 15

f(120) = 240 pesos  (True)

19. Adding and subtracting radicals 2 to the square root of 27 plus 3 to the square root of 75


To add or subtract radicals, we need to first simplify them as much as possible.

Let's start by simplifying the radicals in the expression:

2√27 = 2√(9 x 3) = 2 x 3√3 = 6√3

3√75 = 3√(25 x 3) = 3 x 5√3 = 15√3

Now we can substitute these simplified radicals back into the original expression:

2√27 + 3√75 = 6√3 + 15√3

To combine like terms, we can add the coefficients of the radicals:

6√3 + 15√3 = (6+15)√3 = 21√3

Therefore, the simplified expression is 21√3.

If Emman ride a taxi from his place to the post officewith distance of 400 meters, how much will he pay?2.) If Emman pays Php 75, how many kilometer did hetravel?​


Answer:

1.)47. 00

2.) 2km

Step-by-step explanation:

Emman often ride a taxi from one place to another. The

standard fare in riding a taxi is Php 40 as a Flag-

down rate plus 3.50 for every 200 meters or a fraction of

it.

Questions

1. If Emman ride a taxi from his place to the post office

with distance of 400 meters, how much will he pay?

2.) If Emman pays Php 75, how many kilometer did he

travel?

Formula:

Fare = 40 + (3.5 x (n/200))

Let n be the distance

1. n = 400

Fare = 40 + (3.5 x (400/200))
Fare = 40 + (3.5 x 2)
Fare = 40 + 7
Fare = Php 47

2. To get the distance based on what he paid:

Distance of travel = ((75 - 40)/3.5) x 200
Distance of travel = (35/3.5) x 200
Distance of travel = (10) x 200
Distance of travel = 2000 meters

Converts meters to kilometer by dividing it by 1000 (as 1000m is to 1km)

2000/1000 = 2 kilometers
