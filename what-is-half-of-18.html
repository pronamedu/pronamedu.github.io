1. what is one half times 3 is 18​


The statement "one half times 3 is 18" is not true.

To verify, we can solve for the unknown variable by using basic algebra:

1/2 * 3 = 18

First, we simplify the left side of the equation:

1/2 * 3 = 3/2

So the equation becomes:

3/2 = 18

This is clearly not true since 3/2 is less than 2, and 18 is much greater than 2.

Therefore, the statement "one half times 3 is 18" is false.

A = -18, B = 10

distance = absolute value (A - B)

d = abs (-18 - 10)

d = abs (-28)

d = 28

half distance = d / 2 = 28 / 2 = 14

point at half = A + d/2 = B - d/2

p = A + 14 = B - 14

p = -18 + 14 = 10 - 14

p = -4 = -4
