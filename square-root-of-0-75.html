Square Root Method;

STEP 1. Move the constant to the right and change its sign

[tex]3 {x}^{2} = 75[/tex]

STEP 2. Divide both sides by 3

[tex] \frac{3 {x}^{2} }{3} = \frac{75}{3} \\ \\ {x}^{2} = 25[/tex]

STEP 3. Take the square roots of both sides. Remember to use both positive and negative

[tex] \sqrt{ {x}^{2} } = \sqrt{25} \\ \\ x = + 5 \\ x = - 5[/tex]

Factoring

STEP 1. Divide both sides by 3

[tex] \frac{3 {x}^{2} - 75}{3} = \frac{0}{3} \\ \\ {x}^{2} - 25 = 0[/tex]

STEP 2. Factor the expression

[tex](x - 5)(x + 5) = 0[/tex]

STEP 3. Separate

[tex]x - 5 = 0 \\ x + 5 = 0[/tex]

STEP 4. Solve for X

[tex]x = - 5 \\ x = + 5[/tex]

Completing the square

STEP 1. Move the constant to the right then change the sign

[tex]3 {x}^{2} = 75[/tex]

STEP 2. Divide both sides by 3

[tex] \frac{3 {x}^{2} }{3} = \frac{75}{3} \\ \\ {x}^{2} = 25[/tex]

STEP 3. Since there is no middle term, proceed to square root. use both positive and negative

[tex] \sqrt{ {x}^{2} } = \sqrt{25} \\ \\ x = + 5 \\ x = - 5[/tex]

Quadratic Formula

[tex]x = \frac{ - b + \sqrt{ {b}^{2} - 4ac}}{2a} \\ \\ x = \frac{ - b - \sqrt{ {b}^{2} - 4ac}}{2a}[/tex]

[tex]a = 3 \\ b = 0 \\ c = - 75[/tex]

STEP 1 substitute the variables

[tex]x = \frac{ - (0) + \sqrt{ {(0)}^{2} - 4(3)( - 75)}}{2(3)} \\ \\x = \frac{ - (0) - \sqrt{ {(0)}^{2} - 4(3)( - 75)}}{2(3)}[/tex]

STEP 2 compute

[tex]x = \frac{ + \sqrt{900} }{6} = \frac{ + 30}{6} = + 5 \\ \\x = \frac{ - \sqrt{900} }{6} = \frac{ - 30}{3} = - 5[/tex]

(1/3)(3x²) = (1/75)(1/3)

x² = 1/225

C.  Extract the roots using the square root property where if x² = c, then x=-√c and x=+√c:

x² = 1/225

√(x²) = ±√(1/225)

x = ±(1/15)

x = +15 and x=-15

The roots are +1/15 and -1/15.
