Example A = { the first four odd numbers } B = { the days of the week that starts with the letter T } 3. Set builder notation - is a methof that lists the rules that determine whether an object is an element of the set rather than the actual elements. Example A = { x/x is the first four odd numbers } B = { x/x are the days of the week that starts with the letter T }​
