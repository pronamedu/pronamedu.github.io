30. 23. V How many possible arrangements are there in the letters of the word 'SUNDAY'? ​


Answer:

The word 'SUNDAY' has 6 letters. To determine the number of possible arrangements, we can use the formula for permutations of n objects, which is n factorial (n!).

In this case, we have 6 objects, so the number of possible arrangements is:

6! = 6 x 5 x 4 x 3 x 2 x 1 = 720

Therefore, there are 720 possible arrangements of the letters in the word 'SUNDAY'.

Answer:

720

Step-by-step explanation:

This is a problem about permutations.

6! = 720

! = Factorial, which means that we have to multiply all numbers until number one. In this case is 6*5*4*3*2*1 = 720
