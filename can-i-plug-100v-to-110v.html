8. E=100v I=0.01 find R(COMPLETE SOLUTION)​


OHM'S LAW

Problem:

๑ Given that E = 100 V, I = 0.01, find R.

In this kind of problem

E stands for the VoltageI stands for the Current, and;R stands for the Resistance

Formula:[tex] \underline{ \boxed{ \tt R = \frac{E}{I} }}[/tex]

- To calculate the resistance, divide the given voltage by the given current. The quotient is the answer.

— — — — — — — — — —

Since the givens are

E (Voltage) = 100 V I (Current) = 0.01R (Resistance) = ? (unknown)

Solution:[tex] \tt R = \frac{E}{I} \\ \: \: \: \: \: \: \: \: \: \tt \: = \frac{100}{0.01} \\ \tt \: \: \: \: \: \: \: \: \: \: \: \: \: = \blue{10,000}[/tex]

Therefore, the resistance is 10,000.

10. Find the total power in kW in an unbalanced 3 phase circuit loaded as follows. Phase 1 = 120V, 100A, unity pf. Phase 2 = 100V, 230A, 0.8pf and phase 3 = 110V, 85A, 0.77 pf.


Answer:

Therefore, the total power in kW in the unbalanced 3 phase circuit is 37.6 kW.

Step-by-step explanation:

To find the total power in kW, we need to first find the apparent power (S) in each phase using the formula:

S = V x I

where V is the phase voltage and I is the phase current.

For phase 1,

S1 = 120V x 100A = 12,000 VA

For phase 2,

S2 = 100V x 230A = 23,000 VA

For phase 3,

S3 = 110V x 85A = 9,350 VA

Next, we can find the real power (P) in each phase using the formula:

P = S x pf

where pf is the power factor.

For phase 1,

P1 = 12,000 VA x 1.0 = 12,000 W = 12 kW

For phase 2,

P2 = 23,000 VA x 0.8 = 18,400 W = 18.4 kW

For phase 3,

P3 = 9,350 VA x 0.77 = 7,199.5 W = 7.2 kW (rounded to one decimal place)

Finally, we can find the total power by summing up the real power in each phase:

Total Power = P1 + P2 + P3

= 12 kW + 18.4 kW + 7.2 kW

= 37.6 kW

Therefore, the total power in kW in the unbalanced 3 phase circuit is 37.6 kW.
