15. Which is a benefit of peer-to-peer networking?| A. Easy to add or remove devices. B. Main server has a fast processor. C. More reliable than a client/server network. D. Same software versions on all computers.


Benefit of peer-to-peer networking

The answer to the question is D. Same software versions on all computers.

Peer to peer networking is a type of network where all computers connected to each other, thus there is no server and client. All of the computers are treated equally. Meaning all of the computers in a network has the same software installed, same access, same level of privilege and same settings. Computer in peer to peer network is referred to as a peer.

Meanwhile, in a client server network, there is one computer which could store all the data and has higher specification compared to other computers. This is referred to as a server. All computers connected to it a client which has lower specification and lower privilege compared to a server computer. The advantage of this network set up is when one computer has damage, others will not get affected, except when the server computer gets down.

27. Which of the following is safe practice? Select one: a. Use a combination of letters, numbers, and symbols as a password b. Save your password in every device that you use c. Use your mom's birthday as a password so it's easy to remember d. Write your password in a piece of paper so you won't forget it


The answer is a. Use a combination of letters, numbers, and symbols as a password. Using strong passwords is important in keeping confidential details, and having personal privacy. Therefore, people must use safe practices in making passwords. An example of a safe practice is using a combination of letters, numbers, and symbols as a password.

Why do People Need to Follow Safe Practices when Creating Passwords

Safe practices must be observed by people when creating passwords because of these two main reasons:

to keep confidential and personal information to protect personal privacy against intruders Examples of Safe Practices when Creating Passwords

Aside from using a combination of letters, numbers, and symbols as a password, here are other safe practices when creating passwords:

Do not save your password in every device so that other people who may have access to your device will not be able to open your accounts. Do not use basic and obvious information such as your mom's birthday, your birthday, phone name, names of family members, etc. Do not write password in a piece of paper as other people might find that paper or information. Avoid using same passwords for different accounts. Avoid using passwords using adjacent keys such as "qwerty" or "1234".
