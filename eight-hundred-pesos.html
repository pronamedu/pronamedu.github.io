Solutions:

1. Sixteen is what percent of two hundred?

Base = 200

Percentage = 16%

Rate = Unknown

We can use the formula:

Percentage = (Rate / Base) x 100%

Rearranging the formula, we get:

Rate = (Percentage / 100%) x Base

Substituting the values, we get:

Rate = (16 / 100%) x 200

Rate = 0.16 x 200

Rate = 32

Therefore, 16% of 200 is 32.

2. Find one hundred fifty percent of ninety-eight.

Base = Unknown

Percentage = 150%

Rate = 98

We can use the formula:

Rate = (Percentage / 100%) x Base

Rearranging the formula, we get:

Base = Rate / (Percentage / 100%)

Substituting the values, we get:

Base = 98 / (150 / 100%)

Base = 98 / 1.5

Base = 65.33 (rounded to two decimal places)

Therefore, 150% of 98 is 65.33 (rounded to two decimal places).

3. A cellphone casing is on sale for 85 percent off the regular price. If its original price is forty pesos, how much is the new sale price?

Base = 40

Percentage = 85%

Rate = Unknown

We can use the formula:

Rate = Base - (Percentage / 100%) x Base

Substituting the values, we get:

Rate = 40 - (85 / 100%) x 40

Rate = 40 - 34

Rate = 6

Therefore, the new sale price is 6 pesos.

4. The Calbayog City basketball team won eight percent of their games. If they played twenty-five games, how many games did they win?

Base = 25

Percentage = 8%

Rate = Unknown

We can use the formula:

Rate = (Percentage / 100%) x Base

Substituting the values, we get:

Rate = (8 / 100%) x 25

Rate = 0.08 x 25

Rate = 2

Therefore, the Calbayog City basketball team won 2 games.

5. Eighty is what percent of four hundred?

Base = 400

Percentage = Unknown

Rate = 80

We can use the formula:

Percentage = (Rate / Base) x 100%

Substituting the values, we get:

Percentage = (80 / 400) x 100%

Percentage = 0.2 x 100%

Percentage = 20%

Therefore, 80 is 20% of 400.

6. What is 63% of 100?

Base = 100

Percentage = 63%

Rate = Unknown

Rate = (Percentage / 100%) x Base
