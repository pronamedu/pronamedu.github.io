To find the total number of tires produced in 460 days, we can multiply the number of tires produced per day by the number of days:

6348 tires/day x 460 days = 2,921,680 tires

25. the diameter of a tricycle tire is 60cm. how far will the tire go in one rotation? ​


Answer:

If a tricycle tire travels in one rotation, then the tricycle travels in a distance where d is equal to the circumference of the wheel. This means that we have to use the circumference formula for this problem.

The circumference formula goes like this:

C = 2πr

(where “r” means radius of a circle and “π” is 3.14.)

We can write the following formula for finding the distance traveled by a tricycle when its single wheel makes a complete rotation.

C = 2π(30 cm)

C = 60π

C = 60(3.14)

C = 188.4 cm

Therefore, the tricycle will move 188.4cm in one rotation.
