18. Find the circumference of the circle with a radius of 5 cm. 2. Calculate the circumference of a circle whose diameter is 70 cm. 3. Find the circumference of the circle with a diameter of 8 inches 4. What is the circumference of a circle that has a diameter of 10? 5. The diameter of the wheel is 100 cm. How many revolutions will each wheel make to travel a distance of 157 meters?​


Answer:

1. 31.4cm

2. 219.8cm

3. 25.12 inches

4. 62.8

5. 50 revolutions

Step-by-step explanation:

Formula: C = 2πr

Where:

C = Circumference

π = 3.14

r = radius

diameter to radius is: R = D/2 (where D is the Diameter)

1. C = 2(3.14)(5cm)

C = 31.4cm

2. C = 2(3.14)(35cm)

C = 219.8cm

3. C = 2(3.14)(4inches)

C = 25.12 inches

4. C = 2(3.14)(10)

C = 62.8

5. C = 2(3.14)(50cm)

C = 314cm

To get the number of revolutions of the wheel, divide the distance covered by the circumference of the wheel.

We need to convert 157 meters to cm before dividing, so we multiply 157 by 100 to get 15700 cm.

Therefore,

Number of revolutions = 15700 cm/314 cm

= 50 revolutions.
