7. The sum of five consecutive natural numbers is 145, what is the middle term of these consecutive numbers.​


Answer:

29

Step-by-step explanation:

Let the first natural number be n. We have the other 4 numbers denoted as:

n + 1

n + 2

n + 3

n + 4

Now we will add all of these

n + (n + 1) + (n + 2) + (n + 3) + (n + 4)

We know that the sum of 5 consecutive natural number is 145

n + (n + 1) + (n + 2) + (n + 3) + (n + 4) = 145

We will combine the like terms here:

5n + 10 = 145

5n = 145 - 10

5n = 135

n = 27

n= 27

n+1= 28

n+2= 29

n+3= 30

n+4= 31

These are the other 4 consecutive numbers.

Now if we add all of these up we will get this :

27 + 28 + 29 + 30 + 31 = 145.

The middle term of these consecutive numbers is 29
