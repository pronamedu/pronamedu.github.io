Original rectangle:
W = n
L = n + 2ft
A = n^2 + 2n

New rectangle:
W = n + 1ft
L = n + 3ft
A = n^2 + 4n + 3

Soln.:
n^2 + 2n + 17 = n^2 + 4n + 3
14 = 2n
14/2 = 2n/2
7 = n

Dimensions of the original rectangle:
Width = 7ft
Length = 9ft
