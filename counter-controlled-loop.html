This is performed by adding controllers, compensators, and feedback structures to the system.

Answer:

Typically, we divide feedback loops into two main types:

-positive feedback loops, in which a change in a given direction causes additional change in the same direction. ...

-negative feedback loops, in which a change in a given direction causes change in the opposite direction

25. flowchart or pseudo code that the sum of number 1 up to 1000. Print the value of the counter and the output in each time it enters the loop.


The question says that it has a loop so this is already a clue as to how we should go about making the pseudo code. I usually see two kinds of loops "for", and "while".

First let's try the pseudo code using a for loop:

I'll be putting the actual pseudo code in underline, with explanations in normal font.

Before anything let's initialize our variables first. We need a counter, and a sum, right? So, let's set the following variables:

start

counter = 1, since we start counting at 1

sum = 0, since we haven't started adding yet.

Now, let's go make use the for loop:

for (counter = 1, counter <1001, counter = counter + 1)

The first two arguments say that the loop will continue starting from counter = 1, up to counter <1001 (or <=1000). The third one says that after every iteration (every time it finishes one round of the commands inside the loop), we will add 1 to the counter. This is important for the counter to actually count, and not just stay as 1 forever.

{

sum = sum + counter; This adds the counter to the original sum. We initialized sum to be 0, so in the first iterate, sum = 0+1 = 1.

print (counter); Here, the counter is still 1, so it prints 1

print(sum); Here, sum is already 1 from the sum = sum + counter line, and it prints that too.

}

end

Now what happens during the second iteration:

counter = 2 (from counter = counter + 1)

sum = 1 + 2 = 3 (from previous sum=1 and new counter =2)

And this continues on until 1000.


Here is an example for the while loop, using the same logic as the one in the for loop:

start

counter = 1

sum = 0

while (counter <1001){

sum = sum + counter;

print (counter);

print(sum);

counter = counter + 1;

}

28. 1, what are the three control structures2, what are control structures3, what is sequence 4, what is condition selection5, what is repectition/itertion (loops)​


Answer:

1. The three control structures in programming are sequence, selection, and iteration.

2. Control structures are programming constructs that allow developers to control the flow of execution in a program. They enable the program to take different paths based on certain conditions and repeat certain blocks of code multiple times.

3. Sequence is a control structure that represents the execution of a series of instructions in a specific order. It is the simplest control structure and is used to perform a set of statements one after the other.

4. Condition selection is a control structure that allows the program to make decisions based on some conditions. In other words, it enables the program to take different paths based on whether a certain condition is true or false. The most common way to implement condition selection is by using if-else statements.

5. Repetition/iteration, also known as loops, is a control structure that allows developers to execute a block of code multiple times. Loops are used when a certain set of instructions needs to be repeated multiple times with slight variations. There are three types of loops: for loop, while loop, and do-while loop.
