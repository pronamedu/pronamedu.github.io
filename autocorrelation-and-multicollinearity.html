10. Grace is the sub-editor in a business firm. Which feature of a word processing program would she use to make her changes visible to her senior? A. Track Changes B. Spell Check C. AutoCorrect D. Comments E. Grammar Check


The answer is a. Track Changes

Track Changes is one of the features of word processing software like Microsoft Word and Libre Office Writer that takes note of each change on the document that you are working on from its original state up to the most latest. Businesses uses this to monitor important changes on collaborative documents they are working on to.

To run Track Changes in Microsoft Word, just click on REVIEW menu on the menu toolbar and click Track Changes.
