8. What is the difference between Microsoft Word 2007 and Microsoft Word 2016?​


Huge differences, comparing night and day. But if you’re asking from a financial cost point of view or features? I have to assume the only reason you’re asking is from a cost perspective in that you already own ’07 and want to know if it’s worth it to step up to ’16 or Office 365?

The first and foremost difference would be better, much better security with ‘16.

Then you get into features of ‘16;

’16 can read older files, but ’07 can’t read newer ’16 files e.g., .docx so you need a free “converter” or use a different product to open those with no formatting loss etc.

.docx files have a better chance of opening with everything intact when files get corrupted whereas .doc files Word simply won’t open them

.docx files save space

Better Onedrive integration

Better “Share” and “Co-Authoring”

Bing powered Smart Lookup

The list goes on and on, bottom line if you’re worried about cost… stay where you are. If you have concerns about security or seek better implementation and seamless integration with other products, then go with ’16 or better yet the Office 365 suite.
