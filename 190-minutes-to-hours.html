Answer:

To solve this problem, we can use the formula:

distance = speed x time

First, we need to convert the speed from kilometers per hour to meters per minute, since the time is given in minutes.

75 kilometers per hour = (75,000 meters) / (60 minutes) = 1,250 meters per minute

Now we can plug in the values we have:

distance = speed x time

distance = 1,250 meters per minute x 1 minute

distance = 1,250 meters

Therefore, the car travels 1,250 meters in one minute at a speed of 75 kilometers per hour.
