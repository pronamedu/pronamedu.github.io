30. Is Voy - Vfy equal to Vy? projectile motion btw


What is Voy in projectile motion?

An object launched from the ground with an initial velocity at some angle that lands with zero vertical displacement. ... Also, the vertical displacement is zero, so vfy = -voy. An object launched with an initial velocity at an angle that achieves some vertical displacement.

Projectile motion equations

Horizontal velocity component: Vx = V * cos(α)

Vertical velocity component: Vy = V * sin(α)

Time of flight: t = 2 * Vy / g.

Range of the projectile: R = 2 * Vx * Vy / g.

Maximum height: hmax = Vy² / (2 * g)
