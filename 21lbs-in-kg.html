Step-by-step explanation:

In conversion, it is important to know the Conversion Factor. From Thought.Co, a conversion factor is a number or a formula you need to know in order to convert a measurement in one set of units to the same measurement in another set of units.

In the given, the Conversion Factor is:

[tex]1~\text{lb}=0.45~\text{kg}[/tex]

This conversion factor can be written as a ratio between the units. To do that, simply write it as a rational number.

[tex]\frac{1~\text{lb}}{0.45~\text{kg}}[/tex]: Use this factor if we want to convert [tex]\text{kg}\to\text{lb}[/tex]

[tex]\frac{0.45~\text{kg}}{1~\text{lb}}[/tex]: Use this factor if we want to convert [tex]\text{lb}\to\text{kg}[/tex]

To convert [tex]49~\text{kg}[/tex] into [tex]\text{lbs}[/tex], simply multiply the initial given then multiply it to the conversion factor (In this case, use [tex]\frac{1~\text{lb}}{0.45~\text{kg}}[/tex]). Mathematically:

[tex]49~\text{kg}\cdot \frac{1~\text{lb}}{0.45~\text{kg}}=\frac{49~\text{kg}\cdot 1~\text{lb}}{0.45~\text{kg}} =108.89~\text{lbs}[/tex]

Therefore, when [tex]45~\text{kg}[/tex] is converted to [tex]\text{lbs}[/tex], the result is:

[tex]108.89~\text{lbs}[/tex]
