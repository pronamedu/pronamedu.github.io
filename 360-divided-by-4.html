10. divide 360 pens among 3 pupils in the ratio of 6:4:2.how many pens did each pupil recieve​


Ratio and Proportion

The problem is an application of ratio and proportion. Let's define, ratio and proportion.

What is Ratio

A ratio is an ordered pair of numbers a and b, written as a / b or a : b, where a and b does not equal to zero.

Example

1/2 or 1:2, read as one is to two

3/4 or 3:4, read as three is to four

6/5 or 6:5, read as six is to five

In example number one, it states that in every 1 unit, it has 2 units or item. Like, in a class, in every 1 boy, it has 2 girls in a group presentation. The ratio of boy to girl is 1:2.

Remember that, ratio is always written in lowest term.

What is Proportion

A proportion is an equation in which two ratios are set equal to each other.

Example

2/3 = 4/6

1/3 = 3/9

4/7 = 8/14

You can identify if the set of ratio are proportion when the cross product is equal.

2/3 = 4/6

Cross Product → (2)(6) = (3)(4) → 12 = 12 ✓

In the given problem "Divide 360 pens among 3 pupils in the ratio of 6:4:2. How many pens did each pupil receive?”

Given

360 pens  

3 pupils in the ratio of 6:4:2

Asked

How many pens did each pupil receive?”

Solution

To solve the given problem, first make a linear equation using the principles of ratio and proportion.

Let x is the constant number. The ratio 6:4:2 is equal to 6x:4x:2x, and having a sum of 360.

Then create mathematical equation.

6x + 4x + 2x = 360

12x = 360

X = 360/12

X = 30

Then solve for 6x:4x:2x

6x = 6(30) = 180

4x = 4(30) = 120

2x = 2(30) = 60

Thus, the number of pens for each pupil is 180, 120 and 60.

Check

The sum of pens is 360, so 180 + 120 + 60 = 360, correct.  

Their ratio is 6:4:2 = 180:120:60, correct

Final Answer

The number of pens for each pupil is 180, 120 and 60.
