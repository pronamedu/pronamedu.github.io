8. 972, 324, 108...common ratio​


Ratios

Ratios in Geometry sequences are formulated [tex]\frac{a_{n} }{a_{n-1}}[/tex]

In this problem 972, 324, 108...then the value of

The ratio of the sequence is: [tex]\frac{324}{972} =\frac{108}{324} =\frac{1}{3}[/tex]

Where did it come from? From comparing the second term to the first term, the third term to the second term, the tenth term and the ninth term. ([tex]\frac{a_{3}}{a_{2} } =\frac{a_{2} }{a_{1} } =\frac{a_{10} }{a_{9}}[/tex]). This ratio always remains in the geometric sequence.

Let's practice determining the ratio of the number sequence below.

1. 3,6,12,36... then the ratio=[tex]\frac{6}{3}[/tex]=2

2. 2,6,18,54,… then the ratio=[tex]\frac{6}{2}=3[/tex]

3. 1/2,1/4.1/8,1/16… then the ratio=[tex]\frac{1}{4}:\frac{1}{2} =\frac{1}{2}[/tex]

4. 1/27.1/9.1/3…. then the ratio=[tex]\frac{1}{9}:\frac{1}{27} =3[/tex]
