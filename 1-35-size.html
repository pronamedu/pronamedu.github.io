[tex]\Large\underline{\mathbb{SOLUTION:}}[/tex]

An object is placed in front of a Convex lens 4.5 cm tall at a distance of 10 cm having a focal length of 35 cm. Determine the image distance and image size.

Given values:

[tex]\textsf{\red{Height of object}\:\red{(h)} = \red{4.5 \: cm}}[/tex][tex]\textsf{\blue{Distance of object}\:\blue{(p)} = \blue{10 \: cm}}[/tex][tex]\textsf{\purple{focal length}\:\purple{(f)} = \purple{35 \: cm}}[/tex]

Required:

[tex]{\textsf{\green{Distance of image}}\:\green{(q)} = \: \textsf{\green{Unknown}}}[/tex][tex]\textsf{\pink{Height of image}}\:\pink{(h')} = \: \textsf{\pink{Unknown}}[/tex]

To find the image distance, apply the lens equation:

[tex]\bold{Formula : } \qquad \qquad \qquad \qquad \qquad \qquad \qquad \\ \boxed{ \rm \: \: \: \purple{\dfrac{1}{f}} = \blue{\dfrac{1}{p}} + \green{\dfrac{1}{q}} \: \: \: }[/tex]

Substitute the given values into the lens equation and solve for q:

[tex] \sf: \implies \: {\purple{\dfrac{1}{f}} = \blue{\dfrac{1}{p}} + \green{\dfrac{1}{q}}}[/tex]

[tex] \sf: \implies \: {\purple{\dfrac{1}{35 \: cm}} = \blue{\dfrac{1}{10 \: cm}} + \green{\dfrac{1}{q}}}[/tex]

[tex] \sf: \implies \: { \purple{\dfrac{1}{35 \: cm}} - \blue{\dfrac{1}{10 \: cm}} = \green{\dfrac{1}{q}} }[/tex]

[tex]\sf: \implies \: { \dfrac{\blue{10 \: cm} - \purple{35 \: cm}}{\purple{(35 \: cm)}\blue{(10 \: cm)}} = \green{\dfrac{1}{q}} } [/tex]

[tex]\sf: \implies \: { \dfrac{\blue{ - 25 \: cm}}{\purple{ 350 \: cm^{2} }} = \green{\dfrac{1}{q}} }[/tex]

[tex]\sf: \implies \: { \dfrac{ \cancel{\blue{ - 25 \: cm}}\green{(q)}}{ \cancel{\blue{ - 25 \: cm}}} = \dfrac{\purple{350 \: cm}^{ \cancel{\purple{2}}} }{\blue{ - 25} \: \cancel{\blue{cm}} } }[/tex]

[tex]\sf: \implies \: { \pmb{\green{q} = \green{ -14 \: cm}} }[/tex]

To find the image height, apply the magnification equation:

[tex]\bold{Formula : } \qquad \qquad \qquad \qquad \qquad \qquad \qquad \\ \boxed{ \rm \: \: \: \dfrac{\pink{h'}}{\red{h}} = \dfrac{\green{-q}}{\blue{p}} \: \: \: }[/tex]

Substitute the given values into the magnification equation and solve for h':

[tex]\sf: \implies \: { \dfrac{\pink{h'}}{\red{h}} = \dfrac{\green{-q}}{\blue{p}} }[/tex]

[tex] \sf: \implies \: { \dfrac{\pink{h'}}{\red{4.5 \: cm}} = \dfrac{\green{-( - 14 \: cm)}}{\blue{10 \: cm}} }[/tex]

[tex] \sf: \implies \: { \dfrac{\pink{h'} \cancel{\blue{(10 \: cm)}}} { \cancel{\blue{10 \: cm}}} = \dfrac{\green{63 \: cm}^{ \cancel{\green{2}}} }{\blue{10} \: \cancel{\blue{cm}}} } [/tex]

[tex] \sf: \implies \: { \pmb{\pink{h'} = \pink{6.3 \: cm}} } \\ [/tex]

Therefore, the image distance is -14 cm (negative) and the image height is 6.3 cm.
