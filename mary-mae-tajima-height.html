20 red roses were left in the big flower vase.

Step-by-step explanation:

A big flower vase has 60 roses.

Athena Mae got 2/15 of the 60 red roses. = Athena Mae got 8 red roses.

Mary Grace got 5/15 of the 60 red roses. =  Mary Grace got 20 red roses.

20 + 12 + 8 = 40 red roses were taken.

60 - 40 = 20 red roses were left in the big flower vase.

Word Problem Involving Factoring

In solving word problems, first, identify what is asked. Then, look for the given facts. Establish the number sentence and the operation/s to be used. Make sure that the operation/s used will bring out the correct answer. Check the answer using the number sentence and see if it will satisfy the given condition.

Answers:

a. t = 1 second

b. t = 0.5 second

Solutions:

a. Given: h = -16t² + 8t + 8

by factoring,

-16t² + 8t + 8 = 0

Factor the first term.

-16t² = (-4t)(4t)

Factor the last term.

8 = (4)(2)

Separate the factors and equate them to zero.

(-4t + 4)(4t + 2) = 0

-4t + 4 = 0 ; 4t + 2 = 0

Solve for t.

-4t + 4 = 0                          ;     4t + 2 = 0

-4t + 4 - 4 = 0 - 4               ;     4t + 2 - 2 = 0 - 2

-4t = -4                               ;     4t = -2

[tex]\frac{-4t}{-4}[/tex] = [tex]\frac{-4}{-4}[/tex]                              ;     [tex]\frac{4t}{4}[/tex] = [tex]\frac{-2}{4}[/tex]

t = 1                                    ;      t = - 0.5

Therefore, it will take Trizia 1 second to reach the ground.

b. Given: h = -16t² + 8t + 8

-16t² + 8t + 8 - 8 = 8 - 8

-16t² + 8t = 0

by factoring,

-16t² + 8t = 0

8t(2t - 1) = 0

8t = 0   ; 2t - 1 = 0

Solve for t.

8t = 0            ;  2t - 1 = 0

[tex]\frac{8t}{8}[/tex] = [tex]\frac{0}{8}[/tex]             ; 2t = 1

t = 0             ; t = .5

Therefore, it will take Trizia Mae 0.5 second to be 8 feet above the ground.
