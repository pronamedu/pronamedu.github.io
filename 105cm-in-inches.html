14. The length of a rectangle is8 cm longer than its width.Its area is 105cm?. Find thelength and width of therectangleAnswer:​


Answer:

Length : 15cm

Width: 7 cm

Step-by-step explanation:

Let:

x = width of the rectangle

x + 8 = length of the rectangle

A = area of the rectangle = length * width = 105 cm

Solution:

A = length * width

105 = (x + 8) * (x)

105 = x^2 + 8x

0 = x^2 + 8x - 105

0 = (x + 15) (x - 7)

Then, x = 7 or - 15.

However, there is no such thing as a negative length or width.

Width : x = 7

Length : x + 8 = 7 + 8 = 15

Therefore, the width (x) of the rectangle is 7 cm and the length (x + 8) of the rectangle is 15 cm.
